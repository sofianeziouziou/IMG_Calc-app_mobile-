#app_calc_img

Une application Android pour calculer l'IMG (Indice de Masse Grasse) et donner des conseils santé.

## Fonctionnalités
- Saisie de la taille, poids, âge
- Calcul automatique de l'IMG
- Conseils personnalisés selon sexe et IMG
- Page de login et inscription
